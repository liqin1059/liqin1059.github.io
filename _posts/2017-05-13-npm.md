### npm 安装包
//默认安装最新稳定版本
	npm install package_name@版本号

### 例如：安装Bootstrap

打开powershell到文件夹类目下

	输入nvm use 4.3.0

出现Now using node v4.3.0

	再输入npm install bootstrap

文件夹下面的node_modules即可找到bootstrap

	//初始化操作,给项目添加一个配置文件
	npm init
	（也可以通过 npm init --yes 参数为默认配置）
